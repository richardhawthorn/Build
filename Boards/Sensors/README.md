### Description

Sensor board for use in build system

### Features

- HTU21D (Temperature + Humidity)
- MCP9808 (Temperature)
- MPL3115A2 (Pressure)
- TSL2561 (Light)
- Microphone and amplifier (Sound)
- DIL switch for 4 board addresses
- i2c communication with base
- 3.3v operation
- Surface mount
